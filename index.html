Altman Z'-score — Niet-beursgenoteerde ondernemingen (1983)
Disclaimer: De Altman Z'-score is een statistisch model gebaseerd op historische data. Het is een indicatief hulpmiddel, geen garantie. Andere factoren (sectorspecifieke risico's, marktomstandigheden, managementkwaliteit) zijn niet opgenomen. Gebruik steeds in combinatie met kwalitatieve analyse.
Z' = 0.717·X1 + 0.847·X2 + 3.107·X3 + 0.420·X4 + 0.998·X5
X1 = Werkkapitaal (vlottend − KT schulden incl. 492/3) / Totale activa
X2 = Retained earnings (11+13+14) / Totale activa
X3 = EBIT / Totale activa
X4 = Boekwaarde EV / Totale schulden
X5 = Omzet / Totale activa
